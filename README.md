## FAQ

- **Q: Who can create a tier list post?**  
A: Any subreddit member can create a tier list post.
- **Q: Who can manage items and moderation?**  
A: Tier List Admins manage suggestions, items, and reports. The post creator and subreddit moderators are Admins by default. Additional members can be added as Tier List Admins for each tier list post without having to grant them subreddit moderator permissions.
- **Q: How is the ranking calculated?**  
A: Rankings reflect where the community shows the strongest agreement. Votes are counted per tier, and the tier with the highest vote share is highlighted. In case of a tie between tiers, the item is displayed in the highest-ordered tier while clearly marking the outcome as a split opinion. Full vote distributions and percentages are always available in the item details.
- **Q: How can members suggest new item and categories?**   
A: Members can suggest new items through the suggestion form. If a member selects â€œOtherâ€ for a category, the entered category appears as a suggestion for the Admins.
- **Q: How can a member report an item thats inappropriate or inaccurate**  
A: Members can report items from item details pop-up and voting screen. Admins can then review these reports in the Mod Options and edit or remove the item if needed.
- **Q: How can subreddit moderator ensure appropriate tier list posts are created?**  
A: Just like any other post, the subreddit moderator can approve a post or remove a tier list post using standard moderation tools.
- **Q: For timed voting, can voting window be extended?**   
A: Yes. The Admins can modify the voting time window in the Settings menu by simply choosing a different date and time.
- **Q: Can members see how others voted?**  
A: No. Members can see aggregated community results and thier own private ranking and not other membersâ€™ votes.
